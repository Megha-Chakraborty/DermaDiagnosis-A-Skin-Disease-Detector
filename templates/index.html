About this Project

This project uses machine learning to predict skin disorders, with a focus on melanoma detection. The model was trained on the HAM10000 dataset and applies feature engineering techniques to improve accuracy.

Features analyzed include demographic information, lesion characteristics, and medical history to provide a comprehensive assessment.
